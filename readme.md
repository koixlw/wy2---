 # 物业管理系统 API

基于  xxxx构建的现代化物业管理系统

## 🚀 功能特性

### 📍 住址管理
- 支持小区、栋、楼层、房间的四级树形结构
- 住址的增删改查操作
- 树形结构展示和层级管理
- 软删除机制，保护数据完整性

### 👥 住户管理
- 业主和租户信息管理
- 住户与住址的关联关系
- 住户迁入迁出记录
- 联系方式和身份信息管理

### 💰 费用管理
- 多种费用类型支持（水费、电费、燃气费、物业费等）
- 费用的批量创建和管理
- 缴费状态跟踪
- 费用统计和报表功能
- 逾期费用提醒
